# ユーザー一覧ページ

## 概要
フロントエンドに `/users` ルートを追加し、ユーザー一覧を専用ページで表示できるようにする。
バックエンドの `GET /api/users` とフロントの `UserList` コンポーネントは実装済みのため、ルーティング追加が主な作業。

## 背景・目的
現在すべての機能が1ページに表示されており、URL でユーザー一覧ページに直接アクセスできない。
ルーティングを導入することで各機能に専用 URL を持たせる。

## 機能要件
- `react-router-dom` を導入する
- `/` トップページ: 既存のパネル（Hello, Time, Status）を表示
- `/users` ユーザーページ: ユーザー一覧・追加・削除を表示
- ページ間のナビゲーションリンクを追加

## API仕様
- バックエンド `GET /api/users` は実装済み（変更なし）
- レスポンス: `User[]`（id, name, email, createdAt）

## データモデル変更
なし

## バックエンドのテスト追加
- `UserControllerTest` が存在しないため追加する
  - `GET /api/users` が HTTP 200 を返す
  - `POST /api/users` でユーザー作成できる
  - `DELETE /api/users/{id}` で削除できる

## 受け入れ条件
- [ ] `http://localhost:5173/users` でユーザー一覧が表示される
- [ ] `http://localhost:5173/` でトップページが表示される
- [ ] ナビゲーションリンクでページ間を移動できる
- [ ] `UserControllerTest` が追加されテストがパスする
