# API Contract: Todo API

**Branch**: `001-simple-todo` | **Date**: 2026-03-31
**Base URL**: `http://localhost:8080/api`

## エンドポイント一覧

| Method | Path | 説明 |
|--------|------|------|
| GET | /todos | タスク一覧取得 |
| POST | /todos | タスク作成 |
| PUT | /todos/{id}/toggle | 完了状態のトグル |
| DELETE | /todos/{id} | タスク削除 |

---

## GET /todos

タスクの一覧を取得する。

**リクエスト**: なし

**レスポンス**: `200 OK`

```json
[
  {
    "id": 1,
    "title": "買い物をする",
    "completed": false,
    "createdAt": "2026-03-31T10:00:00",
    "updatedAt": "2026-03-31T10:00:00"
  },
  {
    "id": 2,
    "title": "レポートを書く",
    "completed": true,
    "createdAt": "2026-03-31T09:00:00",
    "updatedAt": "2026-03-31T11:00:00"
  }
]
```

---

## POST /todos

新しいタスクを作成する。

**リクエストボディ**:

```json
{
  "title": "買い物をする"
}
```

| フィールド | 型 | 必須 | 備考 |
|-----------|-----|------|------|
| title | string | ✅ | 空文字・空白のみ不可 |

**レスポンス**: `201 Created`

```json
{
  "id": 3,
  "title": "買い物をする",
  "completed": false,
  "createdAt": "2026-03-31T12:00:00",
  "updatedAt": "2026-03-31T12:00:00"
}
```

**エラーレスポンス**:

| ステータス | 条件 |
|-----------|------|
| `400 Bad Request` | titleが空・空白のみ |

---

## PUT /todos/{id}/toggle

指定したタスクの完了状態を切り替える（完了 ↔ 未完了）。

**パスパラメータ**:

| パラメータ | 型 | 説明 |
|-----------|-----|------|
| id | Long | タスクID |

**リクエストボディ**: なし

**レスポンス**: `200 OK`

```json
{
  "id": 1,
  "title": "買い物をする",
  "completed": true,
  "createdAt": "2026-03-31T10:00:00",
  "updatedAt": "2026-03-31T12:30:00"
}
```

**エラーレスポンス**:

| ステータス | 条件 |
|-----------|------|
| `404 Not Found` | 指定IDのタスクが存在しない |

> **注意**: 旧エンドポイント `PUT /todos/{id}/complete` は削除される。

---

## DELETE /todos/{id}

指定したタスクを削除する。

**パスパラメータ**:

| パラメータ | 型 | 説明 |
|-----------|-----|------|
| id | Long | タスクID |

**レスポンス**: `204 No Content`

**エラーレスポンス**:

| ステータス | 条件 |
|-----------|------|
| `404 Not Found` | 指定IDのタスクが存在しない（現状は無視して204を返す） |

---
